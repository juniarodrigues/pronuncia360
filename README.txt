
Pronúncia360 · Desafio de Sons — Deploy no Netlify
=================================================

Como publicar:
1) Faça login em https://app.netlify.com/
2) Em "Sites" > "Add new site" > "Deploy manually"
3) Arraste esta pasta (ou o ZIP) com o arquivo index.html para o Netlify.
4) Aguarde o deploy e copie o link público gerado (ex.: https://seusite.netlify.app).

Dica: se quiser atualizar, basta enviar um novo ZIP com o index.html revisado.
